# Shemotion website starter

## Run locally

Open the folder in VS Code and run a local server, for example:

```bash
python3 -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```

## Notes
- Hero video has been converted to MP4 for browser support.
- Edit content in `index.html`.
- Main styling is in `styles.css`.
